# ALX Project 0x00: Setup

This project scaffolds a modern web application using **Next.js**, **TypeScript**, **Tailwind CSS**, and **ESLint**.

## Getting Started

1. Clone the repository
2. Run `npm install`
3. Start the development server with: `npm run dev`

## Features

- Project scaffolded using `npx create-next-app`
- ESLint for code quality
- Tailwind CSS for styling
- TypeScript for type safety
